Цей код створює бота, який дозволяє додавати, переглядати, фільтрувати та видаляти справи користувача. 
Використовуємо базу даних SQLite для збереження справ, а aiogram для взаємодії з Telegram API.
Кожна функція та обробник мають відповідні коментарі для пояснення їхньої роботи.


Пояснення:

    config.py:
        Файл для зберігання конфігурації, зокрема токену бота. Ви можете використовувати .env файл для зберігання секретних даних.

    db.py:
        Файл для роботи з базою даних SQLite. Містить функції для ініціалізації бази даних, додавання, отримання, фільтрації та видалення справ.

    bot.py:
        Основний файл бота. Він налаштовує командні обробники для start, add, list, filter та delete.
        Команди /add, /list, /filter та /delete дозволяють користувачам додавати нові справи, переглядати список справ, фільтрувати справи за пріоритетом та видаляти справи відповідно.

Запуск бота
python bot.py
