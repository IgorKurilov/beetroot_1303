система управління нотатками з можливістю створення, редагування, перегляду, видалення, пошуку та фільтрування нотаток за допомогою Django.
модель нотаток models.py
форми для додавання та редагування нотаток forms.py
подання (views) для управління нотатками views.py
URL маршрути urls.py
Шаблон для списку нотаток note_list.html
Шаблон для перегляду та редагування нотатки note_detail.html
Шаблон для додавання та редагування нотатки note_edit.html
Шаблон для підтвердження видалення нотатки note_delete.html

Функціональність фільтрування та пошуку нотаток реалізована у поданні note_list через обробку GET-запитів з параметрами q, category та reminder_after.
